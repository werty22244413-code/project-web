# Интернет-магазин безалкогольных напитков

Адаптивный многостраничный веб-интерфейс интернет-магазина, специализирующегося на продаже крафтовых лимонадов, тонизирующих чаев и безалкогольных вин. Проект выполнен в рамках учебной работы по веб-разработке.

##  Технологический стек
* **HTML5** — семантическая разметка (использование тегов `<header>`, `<nav>`, `<main>`, `<footer>`).
* **CSS3** — построение сеток через Flexbox и CSS Grid (`auto-fit`, `minmax`), кастомные анимации переходов (`transition`, `transform`).
* **Vanilla JavaScript** — реализация бизнес-логики приложения без использования сторонних фреймворков.
* **Web Storage API (LocalStorage)** — хранение данных и состояния корзины на стороне клиента.

##  Структура репозитория
* `index.html` — главная страница проекта (лендинг) с интерактивными карточками категорий товаров.
* `catalog.html` — страница каталога товаров с возможностью динамического добавления позиций в корзину.
* `cart.html` — страница корзины, отображающая выбранные напитки, их количество и автоматически рассчитывающая общую стоимость.
* `style.css` — сквозной файл стилей, обеспечивающий визуальное единство интерфейса и адаптивность.
* `script.js` — скрипт управления состоянием приложения (добавление, удаление, сохранение данных в JSON-формате).

##  Ключевой функционал
1. **Динамический расчет:** Система автоматически калькулирует стоимость как отдельных позиций (цена × количество), так и финальную сумму заказа.
2. **Умная сетка товаров:** Благодаря CSS Grid карточки товаров гибко адаптируются под экраны мобильных устройств, планшетов и десктопов без громоздких медиа-запросов.
3. **Персистентность данных:** Использование связки `JSON.stringify` / `JSON.parse` с `localStorage` гарантирует, что товары в корзине сохранятся даже после перезагрузки страницы или закрытия вкладки браузера.
4. **Интерактивный отклик (UI/UX):** Карточки товаров плавно реагируют на курсор мыши (`translateY` и `scale`), создавая эффект глубины интерфейса.
